# Inception-V3-Keras

### Implementation of Inception V3 convolutional neural network

There is 3 main building block
  - Module A
  - Module B 
  - Module C
  
 and 2 different Grid Size Reduction Blocks.
 
 **The Architecture**
  - Stem
  - 3x Module A
  - Grid Size Reduction
  - 5x Module B
  - Grid Size Reduction
  - 2x Module C
  - Global Average Pool layer
